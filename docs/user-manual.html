OCREditor 官方操作手冊

歡迎使用 OCREditor！這是一款強調「端側 100% 離線運行」的智慧型文件與圖片圖層編輯器。透過領先的 AI 模型，能夠自動為您抽離圖片上的文字，並利用智慧漸層演算填補背景，實現真正的「所見即所改」。

1. 系統需求與安裝

  • Windows: 支援 Windows 10/11，需具備支援 DirectML 之顯示卡。
  • macOS: 支援 macOS 13.0 (Ventura) 以上版本，完美適配 Apple Silicon (M1/M2/M3) 的 CoreML 晶片加速。
  • 安裝方式: 前往官方 GitHub Releases 下載對應的安裝檔，依循系統預設安裝程序即可完成。

2. 基本操作：載入圖片

啟動應用程式後，點擊左上角的 「Load Image」 按鈕，或直接將圖片 (JPG, PNG, TIFF) 拖曳至應用程式視窗中。應用程式會在本地端立即執行 OCR 文字識別與區塊分割。

3. 編輯與背景無痕抹除 (Inpainting)

3.1 修改現有文字

點擊任何被識別出來的文字區塊，輸入您想要的文字。當您開始修改時，系統會自動擷取該區塊四角的背景顏色，使用「四角雙線性漸層演算法」向內渲染背景，完美抹除原有的文字筆畫，達到無殘影的替換效果。

3.2 使用橡皮擦功能（刪除區塊）

若圖上有不想要的浮水印或雜訊被識別為文字，您可以點擊該文字區塊，然後點選 「Delete Layer」（刪除圖層）。文字區塊會消失，且底圖會自動套用漸層背景修補，就像無痕橡皮擦一樣。

3.3 手動框選未辨識區塊

若有文字未被自動辨識，您可以點擊上方的 「Add Region」 按鈕。接著在畫面上使用滑鼠左鍵拖曳，畫出一個新的文字區塊。該區塊同樣會觸發背景修補邏輯，您可以將其當作「手動橡皮擦」或是「新增翻譯區塊」使用。

4. 視圖與工具列

  • 縮放功能 (Zoom): 在右上角的工具列中，您可以使用 ➖ 和 ➕ 微調畫面比例，或者直接拖曳滑桿進行快速縮放。點擊 100% 可快速還原為原始大小。
  • 調色盤 (Color Palette): 點擊彩色圓形圖標，即可開啟系統調色盤，為選定的文字區塊自訂您喜歡的顏色。系統亦提供多組快捷色塊供您快速切換。
  • 全域字型設定 (Global Fonts Settings): 於 Settings 選單中可分別設定「主要 (中日韓)」與「次要 (英數)」預設字型。執行 OCR 後，文字會自動綁定此雙字型組合。
  • 強制套用電腦字型 (Force Computer Font): 啟用此選項後，OCR 辨識完成會立即捨棄圖片原始字型，全面套用您選擇的雙預設字型，維持高度視覺一致性。
  • 字體調整 (Font): 支援系統內建字體列表，並可即時調整字型大小，讓排版更符合您的需求。

5. 儲存與匯出

編輯完成後，點擊上方的 「Save Image」 即可將目前畫面包含替換的文字與修復的背景合併，匯出為一張完整的圖檔。所有處理皆在您的設備上完成，完全確保機密文件的安全。
