隱私權與資料保護政策

適用於 Windows (WPF/DirectML) 平台

本隱私權政策旨在說明 OCREditor（以下簡稱「本軟體」）在 Windows 平台環境下，如何處理及保護您的資料。我們深知商業機密與個人隱私的重要性，因此本軟體的核心架構即是以「100% 本地端離線處理」為最高原則設計。

1. 資料的收集與處理

  • 完全離線執行： 本軟體在 Windows 10/11 上運作時，所有核心功能（包含 OCR 文字辨識、LaMa 背景抹除、圖片裁切與文字渲染）均利用 Windows AI Platform 與 DirectML 技術，完全在您的本機顯示卡與 CPU 上進行。
  • 零伺服器傳輸： 我們「不會」將您載入的任何圖片、解析出的文字、或編輯後的成品上傳至任何雲端伺服器。
  • 無追蹤程式碼： 本軟體未內嵌任何廣告 SDK 或背景數據收集（Telemetry）工具，我們不會追蹤您的使用習慣或收集您的硬體識別碼。

2. 暫存檔案管理

當您在 Windows 上使用本軟體時，為維持軟體的正常運作，系統可能會在您的使用者暫存資料夾（例如 %TEMP% 或 AppData\Local\OCREditor）內產生暫時性的快取檔案：

  • 記憶體層級處理： 大多數的圖片資料皆於隨機存取記憶體 (RAM) 中即時運算，並未寫入硬碟。
  • 歷史紀錄： 為了提供「復原 (Undo)」功能，軟體會暫時在記憶體中存放操作紀錄，當您關閉軟體時，這些紀錄會自動銷毀。

3. 權限需求說明

在 Windows 平台上，本軟體僅需極基本的使用者權限即可運行，主要包含：

  • 本地檔案存取： 僅用於讀取您選擇載入的圖片，以及儲存您匯出的編輯完成檔案。
  • GPU 加速權限： 為了呼叫 DirectML 進行高效能的 AI 運算，軟體會存取您顯示卡的運算資源，但不會讀取與軟體無關的顯示記憶體內容。

4. 隱私權政策的修訂

本隱私權政策將隨軟體版本更新而可能進行微調，所有修改將即時發布於本頁面。我們承諾「完全不將使用者資料外洩至網際網路」的原則絕不改變。
