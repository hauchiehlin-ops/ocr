隱私權與資料保護政策

適用於 macOS (CoreML/Metal) 平台

本隱私權政策旨在說明 OCREditor（以下簡稱「本軟體」）在 macOS 系統環境下，如何處理及保護您的資料。本軟體嚴格遵循 Apple 的隱私權指導原則，致力於保障您的資訊安全。

1. 100% 本機端處理

  • CoreML 神經網路引擎： 本軟體針對 Mac 設備（特別是 Apple Silicon M系列晶片）進行深度最佳化，所有圖像識別、OCR 與智慧背景塗抹運算，均透過設備本機的 CoreML 框架及 Neural Engine 執行。
  • 不依賴網路連線： 我們保證絕不將您載入的任何圖像檔、機密文件或編輯結果上傳至任何外部伺服器或 iCloud 空間（除非您自行將檔案存入 iCloud Drive）。

2. macOS 沙盒與權限保護

本軟體遵守 macOS 的 App Sandbox 規範：

  • 檔案存取限制： 我們僅能存取您透過「開啟檔案」對話框主動選擇的圖片，或您透過拖放 (Drag & Drop) 方式放入應用程式的檔案。未經您的授權，本軟體無法掃描或讀取您磁碟機中的其他檔案。
  • 無背景監控： 本軟體絕不會存取您的麥克風、攝影機、位置資訊、通訊錄或剪貼簿（除非您主動貼上文字），也絕無包含任何第三方的追蹤或廣告模組。

3. 崩潰報告與診斷資訊

如果您在 macOS 上遇到程式崩潰，Apple 作業系統可能會詢問您是否要傳送崩潰報告給開發者。此報告僅包含系統狀態與堆疊追蹤資訊（Stack Trace），絕不包含您正在編輯的圖片內容或文字。您可以隨時在 macOS 系統設定的「隱私權與安全性」中關閉此功能。

4. 政策更新

我們將持續確保產品符合最嚴格的隱私標準，本聲明如有更新將立即公佈於此頁面。
